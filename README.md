# 📚 مكتبة بر الوالدين للعلوم الشرعية

منصة علمية دعوية تعليمية متخصصة في نشر العلوم الشرعية بمنهج أهل السنة والجماعة، بأسلوب عصري وتفاعلي.

![Cover](assets/images/cover.png)

## ✨ المميزات
- **إدارة المشايخ والعلماء:** قسم خاص لعرض سير المشايخ ودروسهم ومواقع تواصلهم.
- **جدول الدروس الأسبوعي:** نظام تتبع للدروس والمحاضرات مع دعم لعدة مشايخ في الدرس الواحد.
- **تكامل مع Blogger:** جلب أحدث المقالات العلمية تلقائياً من المدونة.
- **معرض مرئي:** عرض الدروس المسجلة من يوتيوب بشكل منظم.
- **لوحة تحكم كاملة:** واجهة سهلة للمدير لإدارة كافة محتويات الموقع وتحديث البيانات لحظياً.
- **دعم Offline:** يعمل الموقع كـ PWA (Progressive Web App) ويدعم العمل بدون إنترنت جزئياً.

## 🛠️ التقنيات المستخدمة
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6).
- **Backend/Database:** Firebase Realtime Database.
- **Storage:** Firebase Storage (لرفع صور المشايخ).
- **Integration:** Blogger API / RSS.
- **PWA:** Service Workers & Web Manifest.

## 📁 هيكلية المشروع
```text
mktabat-birwalidin/
├── assets/
│   ├── images/         # الشعارات وصور الغلاف والأيقونات
│   └── docs/           # المستندات التوضيحية
├── css/
│   ├── style.css       # التنسيقات الرئيسية للموقع
│   └── admin.css       # تنسيقات لوحة التحكم
├── js/
│   ├── script.js       # منطق الموقع الرئيسي
│   └── admin.js        # منطق لوحة التحكم والربط مع Firebase
├── index.html          # الصفحة الرئيسية للموقع
├── admin.html          # لوحة التحكم الخاصة بالمدير
├── manifest.json       # إعدادات الـ PWA
├── sw.js               # Service Worker للعمل بدون إنترنت
├── robots.txt          # إعدادات محركات البحث
└── README.md           # هذا الملف
```

## 🚀 كيفية البدء
1. قم برفع الملفات إلى أي استضافة تدعم المواقع الاستاتيكية (مثل GitHub Pages أو Firebase Hosting).
2. تأكد من إعدادات الـ Firebase في ملف `js/admin.js` و `js/script.js` (مفاتيح الـ API).
3. للوصول للوحة التحكم، توجه إلى `your-domain.com/admin.html`.
